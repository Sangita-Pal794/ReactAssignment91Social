# 91-Social-assignment

91 Social Assignment

Below are the instructions for _developing_ the ReactAssignment repo:

## Getting started

```shell
npm install # configures all the child packages
npm start # for running assignment
```
Features:
1. Fetch the data in the APIs and display them.
2. Pagination implemented for both history and address details.
3. Search functionality(For Text And Number) for both history and address details.
4. React router for navigation.
5. Reusable react components made.
